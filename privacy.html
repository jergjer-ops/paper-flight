Политика конфиденциальности

Эта политика объясняет, какие данные обрабатывает игра «Бумажный полёт», зачем они нужны и как пользователь может запросить их удаление.

Дата вступления в силу: 19 июля 2026 годаОператор: jergjer-ops

1. Сфера действия

Политика применяется к Telegram-боту игры, Telegram Mini App и веб-версии игры «Бумажный полёт» (далее — «Игра»). Игра является независимым сервисом и не управляется Telegram.

2. Какие данные мы обрабатываем

При запуске через Telegram

  • числовой Telegram ID;
  • отображаемое имя и, если доступно, имя пользователя;
  • код языка Telegram;
  • подписанные Telegram данные запуска, используемые для проверки личности пользователя.

Игровые данные

  • лучший результат, количество полётов и последние результаты;
  • игровые сессии, время начала и завершения, начисленные монеты;
  • выбранные и приобретённые самолётики;
  • идентификатор уникального посетителя для общего счётчика.

Данные на устройстве

В локальном хранилище браузера могут сохраняться имя профиля, результаты, монеты, выбранный самолётик и настройка музыки. Эти данные остаются на устройстве, пока пользователь не очистит данные сайта или приложения.

3. Для чего используются данные

  • для входа без отдельного пароля и защиты имени от подмены;
  • для сохранения прогресса и отображения таблицы лидеров;
  • для подсчёта уникальных игроков;
  • для проверки правдоподобности результатов и предотвращения злоупотреблений;
  • для обеспечения стабильности и безопасности Игры.

Мы не продаём персональные данные и не используем их для сторонней рекламы или кредитного скоринга.

4. Где обрабатываются данные

  • Telegram передаёт Mini App данные запуска и управляет интерфейсом бота;
  • Supabase хранит проверенные профили, игровые сессии, результаты и агрегированную статистику;
  • GitHub Pages размещает статические файлы Игры и этой политики.

Эти поставщики могут обрабатывать техническую информацию, например IP-адрес и журналы запросов, согласно собственным политикам конфиденциальности.

5. Публичные данные

В общей таблице лидеров публично отображаются имя игрока, лучший результат и отметка подтверждённого Telegram-профиля. Telegram ID публично не показывается.

6. Срок хранения

Профиль, лучший результат, игровые сессии и идентификатор посетителя могут храниться, пока работает Игра или пока пользователь не запросит удаление. Локальные данные хранятся на устройстве до очистки данных сайта. Технические записи защиты от повторной доставки webhook хранятся до 7 дней, а краткосрочные записи ограничения частоты запросов — обычно не более суток. Резервные копии поставщиков могут очищаться позже в рамках их обычного цикла обслуживания.

7. Безопасность

Передача данных выполняется по HTTPS. Telegram-данные запуска проверяются на серверной стороне. Доступ к записи результатов ограничен, а игровые сессии используются однократно. Ни один способ защиты не гарантирует абсолютную безопасность, но мы применяем разумные технические меры для снижения риска.

8. Права пользователя и удаление

Для немедленного удаления серверного профиля, результатов, игровых сессий, участия в вызовах и записи счётчика откройте личный чат с действующим ботом Игры и отправьте /delete_me, затем /delete_me confirm. Команда принимается только от владельца соответствующего Telegram ID и не требует публикации идентификатора.

Для запроса сведений или исправления имени используйте страницу проекта. Не публикуйте Telegram ID и другие личные данные в открытом обращении: сначала попросите предоставить приватный способ подтверждения личности.

Локальные данные можно удалить самостоятельно, очистив данные сайта в браузере или данные Mini App в настройках Telegram.

9. Дети

Игра не запрашивает дату рождения и не предназначена для осознанного сбора чувствительных данных детей. Родитель или законный представитель может запросить удаление данных тем же способом.

10. Изменения политики

Политика может обновляться при изменении функций Игры или требований законодательства. Новая версия публикуется по этому же адресу с обновлённой датой.

Использование Игры также регулируется условиями и политикой конфиденциальности Telegram. Эта страница описывает обработку данных именно оператором «Бумажного полёта».

Privacy Policy — English summary

Paper Flight processes a verified Telegram user ID, display name, language, game scores, flight sessions and a visitor identifier to authenticate players, save progress, provide the leaderboard, count unique players and prevent abuse. Public leaderboard entries show the player name and best score but never expose the Telegram ID.

Data is processed using Telegram, Supabase and GitHub Pages. It is not sold or used for third-party advertising. Server data may be retained while the Game operates or until deletion is requested; local data remains until site data is cleared.

To delete server-side game data, open a private chat with the active Game bot and send /delete_me, followed by /delete_me confirm. For access or correction requests, contact the operator through the project page. Never post private identifiers publicly.
